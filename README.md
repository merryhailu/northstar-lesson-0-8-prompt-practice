## Project Overview

This exercise focuses on using the five prompt techniques and comparing the outputs from different AI models. The project was tested using Claude and OpenAI’s ChatGPT to analyze how structured prompts improve the quality of AI-generated responses.

The project also includes verification of the generated code outputs using sample data to ensure the results are correct.

## Objectives

- Learn and apply the five prompt techniques
- Compare outputs from different AI tools
- Understand how structured prompts improve AI responses
- Verify generated code manually using sample data

## AI Tools Used

Claude
ChatGPT

## Verification Process

The generated code was tested using sample customer spending data.

## Verification Results

Customers 9 and 10 were identified as the highest spenders.
The results confirmed that the code worked correctly with the sample dataset.
